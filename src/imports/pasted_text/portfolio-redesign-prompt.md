## 6. MOTION PRINCIPLES

Use motion only where it supports clarity.

Motion style:
- subtle
- fast
- elegant
- no bouncing
- no childish effects
- no distracting loops

Timing:
- hover: 150–250 ms
- page/section transition: 350–500 ms
- reveal animation: 300–400 ms
- easing: ease-out or gentle cubic

Use Figma Smart Animate where possible.

Motion hierarchy:
1. Navigation states
2. Card hover
3. Hotspot reveal
4. Flow step reveal
5. CTA hover

Do not create heavy animation that harms PDF export.
