### 3.3 Typography System

Use a premium typography system.

Primary font:

- Inter, Satoshi, Manrope, Geist or similar modern sans-serif.

Optional accent font:

- Use a restrained editorial serif only for very large section numbers or short display accents.
- Do not use serif for body text.

Type scale:

- Huge display: 96–120 px
- Cover title: 80–104 px
- Case opener title: 72–96 px
- Page title: 52–64 px
- Section title: 34–44 px
- Card title: 24–30 px
- Body: 19–23 px
- Caption: 15–17 px
- Metadata: 13–15 px

Typography style:

- confident
- large
- clean
- slightly editorial
- strong contrast between headings and body
- short line lengths
- high readability

Avoid:

- centered generic title pages
- all caps everywhere
- tiny captions
- too many font weights
- weak visual hierarchy
